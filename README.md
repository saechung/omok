# 🎮 Go로 만든 콘솔 오목 게임 (19x19)

Go 언어를 사용하여 터미널(콘솔) 환경에서 실행되는 2인용 19x19 오목 게임입니다.
흑돌(●)과 백돌(○)이 번갈아 가며 착수하며, 흑돌은 3*3을 둘 수 없습니다.

---
